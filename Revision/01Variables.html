Variables in JS

variables have 3 types

  1. var
  2. let
  3. const
  1. Var => var name = "jyoti" ; Declaration and Assignment
    var name = "jyoti kumawat" ; Re-Declaration is posibile
    name = "jyoti kumawat frontend developer"; Re-Assignment

  2. let => let name = "jyoti"; declaration and assignment posible but re-declaration is not posible
    name = "jyoti kumawat"; re-assignment posible

  3. const => const name = "jyoti"; decration assignment posible
    name = "jyoti kumawat" ; re-assignment is not posible
    const name = "jyotikumawat"; Re-Declaration is not posibile
